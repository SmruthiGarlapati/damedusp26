Section A — Short Answer

Problem 1 5 pts

What is the output of the following code?

Node* head = new Node(10);
head->next = new Node(20);
head->next->next = new Node(30);

Node* cur = head;
while (cur) {
    cout << cur->val << " ";
    cur = cur->next;
}
delete head;
Answer: 10 20 30 — traverses from head to nullptr printing each value. Note: delete head only frees the first node — this is a memory leak! Should traverse and delete each node.
Problem 2 5 pts

Given a BST with root 5, and insert order: 3, 7, 1, 4, 6, 8. Draw the tree and give the in-order traversal output.

Answer: In-order: 1 3 4 5 6 7 8 (always sorted for a BST). Tree structure: 5 is root, left subtree has 3 (with children 1, 4), right subtree has 7 (with children 6, 8).
Problem 3 5 pts

What is the time complexity of deleting a node from a singly linked list given a direct pointer to that node (not a value to search for)?

Answer: O(n) — even with a direct pointer to the node, you must traverse from head to find the previous node to update its next. (For a doubly linked list the answer would be O(1).)

Section B — Tracing

Problem 4 10 pts

Trace the following recursive function on the BST with root=8, nodes: 4, 12, 2, 6, 10, 14.

int mystery(Node* n) {
    if (!n) return 0;
    return 1 + max(mystery(n->left), mystery(n->right));
}

What does mystery(root) return? What does this function compute?

Answer: Returns 3. This is the height function — counts the number of nodes on the longest root-to-leaf path.
Problem 5 10 pts

What is the pre-order traversal of a BST built by inserting: 5, 2, 8, 1, 3, 7, 9?

Answer: 5, 2, 1, 3, 8, 7, 9 — pre-order visits Root first, then recursively left, then right subtree.

Section C — Code Writing

Problem 6 15 pts

Write a function int sumList(Node* head) that returns the sum of all values in a linked list.

Answer: 
int sumList(Node* head) {
    int total = 0;
    while (head) {
        total += head->val;
        head = head->next;
    }
    return total;
}
Problem 7 15 pts

Write a function bool isBST(Node* n, int min, int max) that returns true if a binary tree is a valid BST.

Answer: 
bool isBST(Node* n, int mn = INT_MIN, int mx = INT_MAX) {
    if (!n) return true;
    if (n->val <= mn || n->val >= mx) return false;
    return isBST(n->left,  mn, n->val) &&
           isBST(n->right, n->val, mx);
}
